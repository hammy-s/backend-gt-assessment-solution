# Task Management API

A simple REST API built with Node.js and Express for managing tasks with user assignment.  
This project demonstrates input validation, authorization, and clean code structure.

---

- [npm](https://www.npmjs.com/) (comes with Node.js)
- (Optional) [Postman](https://www.postman.com/) or `curl` for testing
